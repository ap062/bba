# bba-v1.0
bug bounty automation 
prerequisites
 > subfinder
 > assetfinder
 > amass
 > shuffledns
 > httpx
 > nuclei
 > waybackurls
 > ffuf
 > gf 
 > unfurl
 > massdns
 
 
<h1>Installation </h1> 

git clone  https://github.com/ap062/bba    

cd  bba

chmod 777 bba.sh


<h1> Usage: </h1>
  ./bba.sh target.com
  
  
  It will do basic recon of your target
